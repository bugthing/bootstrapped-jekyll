Git to Deploy

Here you can gain access to information relating to Jira, BitBucket and any deployed containers. 

git --> hook fires      --> write commit id (plus other info) to store
                           --> emit event: BBHOOK_RECIEVED(commit_id)

get event: BBHOOK_RECIEVED --> start build
                           --> write to store

get event: BBHOOK_RECIEVED --> get jira info
                           --> write to store

get event: BBHOOK_RECIEVED --> get bitbucket info
                           --> write to store

http --> check store    --> output data
